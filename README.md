🚀 Space Memory Adventure

¡Bienvenido a Space Memory Adventure! Un juego de memoria galáctico diseñado para poner a prueba tu concentración y agilidad mental mientras exploras los confines del universo.
Este proyecto es una aplicación web interactiva, moderna y totalmente adaptada para dispositivos móviles, ideal para formar parte de un portfolio de desarrollo frontend.

🎮 Cómo Jugar

1.El Objetivo: Encuentra todas las parejas de iconos espaciales en el menor número de intentos posible.
2.Mecánica: Haz clic o toca una carta para voltearla y revelar su icono. Luego, elige otra carta.
3.Acierto: Si los iconos coinciden, las cartas permanecerán visibles.
4.Fallo: Si no coinciden, se volverán a ocultar tras un breve segundo.
5.Victoria: ¡La misión termina cuando logras emparejar las 16 cartas del tablero!

✨ Características Principales

📱 Diseño Totalmente Responsivo: Optimizado para una experiencia fluida tanto en computadoras de escritorio como en smartphones y tablets.
🎨 Interfaz Moderna: Construido con Tailwind CSS, utilizando gradientes, efectos de cristal (backdrop-blur) y una paleta de colores "Space Dark".
🔄 Animaciones 3D: Efectos de volteo de cartas realistas mediante transformaciones CSS3.
💾 Sin Instalación: Funciona directamente en el navegador utilizando tecnologías web estándar.
⚡ Alto Rendimiento: Código ligero y eficiente escrito en JavaScript Vanilla (JS Puro).

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica y elementos de juego.
CSS3 (Tailwind CSS): Estilos modernos, diseño responsivo y animaciones 3D.
JavaScript (ES6+): Lógica del juego, manipulación del DOM y gestión de estados.
